# 🌐 Singli-Translate Hub

**Singli-Translate Hub Pro** is a high-performance, modern web application designed to bridge the language gap between Sinhala and English. It features a custom phonetic transliteration engine that allows users to type in Singlish and receive instant, accurate results in both Sinhala and English, all within a beautiful **Glassmorphism** interface.

---

## ✨ Key Features

* **🔄 Dual-Mode Translation:**
    * **Singlish Mode:** Type phonetically (e.g., "Oya kohomada") to get Sinhala and English results simultaneously.
    * **Reverse Mode:** Direct English-to-Sinhala translation for quick meaning lookups.
* **🔊 Text-to-Speech (TTS):** Integrated voice synthesis to hear the correct pronunciation of English translations.
* **📜 Smart History:** Automatically tracks and displays your last 5 translations using browser LocalStorage—no database required.
* **📖 Interactive Mapping Guide:** A built-in modal guide to help users master Singlish-to-Sinhala phonetic mapping (e.g., `gh` ➔ `ඝ`, `aa` ➔ `ආ`).
* **🎨 Modern Glassmorphism UI:** A sleek, dark-themed interface with blurred backgrounds, vibrant gradients, and smooth CSS animations.
* **⚡ Real-Time Processing:** Instant transliteration with a dynamic loader for API-based translations to ensure a smooth user experience.
* **📱 Fully Responsive:** Works perfectly across desktops, tablets, and smartphones.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Advanced Gradients & Animations).
* **Logic:** Vanilla JavaScript (Phonetic Mapping Algorithm).
* **API:** Google Translate API (Integration for high-accuracy translation).
* **Storage:** Browser LocalStorage API (For History persistence).
